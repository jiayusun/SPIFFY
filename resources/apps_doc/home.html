SMILI Assistant

Applications

The SMILI documentation for applications can be found on this page. Apart from the sMILX main application in SMILI, smaller command-line applications for deployment in cloud-like or other environments are also provided. These include:
  1. milxImageApp - An image processing application for 3D image volumes. Get info, mask, threshold, smooth etc. images.
  2. milxModelApp - A surface/model processing application. Smooth, register, operate on scalars etc. of meshes.
  3. milxOverlay - A (off screen rendering) screenshot application that well suited for taking snapshots of processed data for quick visual inspection of results or display on websites.

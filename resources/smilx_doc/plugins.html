Plugins

This page lists the documentation available for the various sMILX plugins. Plugins are generally libraries that are added at run-time to enable extra features within the application. There are in plugin form because either rely on other external libraries, have different software licenses or enable non-standard or specialised features. The current list of plugin documentations are as follows:
  1. DICOM - View, convert and anonymise DICOM images
  2. SSM - Statistical shape model visualisation and analysis, including focused shape modelling.
